# Domastroi Price Bot

Telegram бот для расчета стоимости ремонта компании Domastroi.

## Локальная установка

1. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
# Отредактируйте .env и укажите свои значения
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите бота:
```bash
python bot.py
```

## Деплой на VPS

См. подробную инструкцию в [DEPLOY.md](DEPLOY.md)

Быстрый старт:
```bash
chmod +x deploy.sh
sudo ./deploy.sh
```

## Функционал

- Пошаговый опрос пользователей (5 вопросов)
- Инлайн-кнопки для выбора вариантов
- Валидация ввода площади (только цифры)
- Запрос номера телефона через Telegram
- Автоматическое удаление старых сообщений
- Отправка заявок в админский чат
- Админ-панель для загрузки документа о ПД

## Команды

- `/start` - начать работу с ботом
- `/admin` - админ-панель (только для админов)

## Админ-панель

Админы могут загружать фото или PDF документ с согласием на обработку персональных данных, который будет автоматически прикрепляться к соответствующему сообщению.

## Конфигурация

Все настройки хранятся в файле `.env`:
- `BOT_TOKEN` - токен бота от @BotFather
- `CHAT_ID` - ID чата для отправки заявок
- `ADMIN_IDS` - ID админов через запятую
